# Avoiding-Poop
 ### 1. 프로젝트명
           * openCV와 학습 데이터를 사용한 얼굴 감지 마스킹 프로그램
  
 ### 2. 프로그램 설명 
           * SDL을 사용한 게임 데이터(캐릭터, 똥) 로딩 및 렌더러 생성
           * 좌표 설정을 이용한 캐릭터의 이동 검사 진행

 ### 3. 사용 언어
           * C++, SDL(Simple Directmedia Layer)
